How to Answer Questions?

Step 1: Initial Screen displayed

The initial screen displays the map on top with some quiz points as pink markers and the user’s location in blue marker(Fig 1).


Figure 1: Shows question points in pink and the user’s current location in blue.

Step 2: Answering the question

When the user is close enough to the question point a pop-up is triggered to the user’s closest question (Figure 2). The answer is chosen by clicking one of the radio buttons (Figure2).

Figure 2: Shows pop-up message to the user closest question(left) and user selected answer (right).

Step 3: Outcome of the question

Once the answer is selected and the submit answer button is pressed, the pop-up message closes and the quiz point turns green or red depending if the answer provided was right (green) or wrong (red) (Figure 3).


Figure 3: Shows the outcome once the question is answered, the wrong answer (red), the correct answer (green) and yet to be answered (pink).

App Functionalities

Quiz App Features

Menu-based options are composed of five functionalities (Figure 4 and Table 1) that are described as follow:

  1. Your Points: shows the number of correctly answered questions by the user;

  2. Your Ranking: shows the user ranking position in relation to any other user.

  3. Show top 5 scorers: shows the five best-ranked users;

  4. Add Five question closer to me: adds a map layer with the five closest questions added by any user in relation to the user's current position;

  5. Remove Five questions closer to me: removes the map layer with the five closest questions added by any user in relation to the user's current position returning to the normal viewing


Figure 4: Shows left side menu functionalities.
